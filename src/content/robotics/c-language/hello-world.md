# Hello World

欢迎来到机器人和嵌入空间。学习每一门语言的起点通常都是 `Hello World`。

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

## 为什么是 printf？
在这段代码中，`stdio.h` 包含了标准输入输出库，而 `printf` 就是其中进行格式化输出的函数。
